# cooking-recipes-app


This is a test project to demonstrate how Claude Code can be used to implement a web application E2E.

# Main requirements 
- objective provided in spec/objective.md and remain fixed
- all the other choices are made by Claude Code and corresponding sub-agents
- Spec Driven Development approach is used to avoid hallucinations and achieve effective use of tokens
    - https://developer.microsoft.com/blog/spec-driven-development-ai-native-engineering
    - https://github.com/github/spec-kit
